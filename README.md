- Visualizes 5 sorting algorithms:

- Built using `Tkinter`
- Interactive GUI with dropdown menu
- Real-time color-based comparisons and swaps

##  Tech Used

- Python
- Tkinter (for GUI)
- Core DSA (Sorting)
